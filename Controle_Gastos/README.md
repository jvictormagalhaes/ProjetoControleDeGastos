README:

Aplicação Front-end desenvolvida utilizando React com TypeScript.
O projeto utiliza Vite como ferramenta de build e TailwindCSS para estilização.

Definição das rotas utilizadas ficam em: App.tsx;
Roteamento das páginas foi feito com React-Router; 
Consultas para API feitas com Axios;

Como rodar o Projeto:
Instale as dependências: npm install
Execute o projeto: npm run dev

Configuração da API:
Arquivo: Serives/api.ts; 
URL da API: baseURL:"https://localhost:7297"
