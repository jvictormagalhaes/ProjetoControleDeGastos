API Controle de Gastos feita em .NET utilizando EntityFramework Core e Swashbuckle.AspNetCore para gerar 
a documentaçăo via Swagger ao rodar a aplicaçăo.

Definiçăo da String de Conexăo com o Banco está no documento ""appsettings.json";

Banco de Dados utilizado foi MySQL:
Banco = ControleGastos;
Usuario = root;
Senha = root
